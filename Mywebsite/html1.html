What is web (www) ?

The world wide web was originally design in 1991 by Tim Berners-Lee in switzerland. www means interlinking of webpages. webpages are written in HTML (Hyper Text Markup Language). www is also called internet of the web. all the webpages, video, images are accessed via a web broweser on the internet with use of URL (Uniform Resource Locater)

What is Html ?

Html stands for hyper text markup language. it is the main language which is used to create a web pages on the internet. < > are used in html, so this is called markup language. it is developed in 1991 by Tim Burner lee.
